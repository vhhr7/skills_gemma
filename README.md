# skills_gemma

Colección de skills para [AI Edge Gallery](https://github.com/google-ai-edge/gallery) con Gemma 4.

## Skills disponibles

| Skill | Descripción |
|-------|-------------|
| [screenshot-to-calendar](./screenshot-to-calendar/) | Crea eventos en Google Calendar desde un screenshot o texto |

## Cómo cargar un skill por URL

En AI Edge Gallery: **Agent Skills → "Skills" → (+) → "Load skill from URL"**

Pega la URL de GitHub Pages apuntando a la carpeta del skill, por ejemplo:
```
https://vhhr7.github.io/skills_gemma/screenshot-to-calendar
```
